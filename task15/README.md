# WB_project_2.15


Для компиляции утилиты выполните команду в этой директории:
```
make build
```
Для запуска тестов:
```
bash test.sh
```
Можно запускать вручную через `./GShell` и так же через make `make run`.

Для проверки кода выполните следующие команды:
```
golangci-lint run ./...
go vet ./...
```
