# WB_project_2.13


Для компиляции утилиты выполните команду в этой директории:
```
make mycut
```
Для запуска тестов:
```
bash test.sh
```
Можно запускать вручную через `./mycut` и так же как обычный `grep`.

Для проверки кода выполните следующие команды:
```
golangci-lint run ./...
go vet ./...
```
